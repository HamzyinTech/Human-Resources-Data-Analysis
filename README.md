HR Analytics Dashboard (Excel)
Project Overview

This project focuses on analyzing employee data to uncover key insights related to employee performance, salary distribution, and workforce demographics. The analysis was conducted using Microsoft Excel, with a focus on data visualization and interactive dashboards to support HR decision-making.

The goal of this project is to demonstrate how Excel can be used as a powerful tool for data analysis, reporting, and insight generation in HR analytics.

Objectives
Analyze employee performance metrics.

Understand salary distribution across different employee groups.

Examine workforce demographics such as gender, department, and job roles.

Identify patterns and trends that can support HR decision-making.


Tools & Technologies

Microsoft Excel

Pivot Tables

Pivot Charts

Data Visualization

Interactive Dashboards


Data Processing

The dataset was cleaned and prepared before analysis by:

Removing inconsistencies and duplicate records

Formatting data types

Handling missing values

Structuring data for pivot table analysis


Dashboard Features

The Excel dashboard provides interactive visual insights including:

Employee Performance Analysis

Salary Distribution Overview

Workforce Demographics Breakdown

Department-wise Employee Distribution

Interactive Charts and Visualizations

These visualizations help highlight key trends and patterns within the HR dataset.


Key Insights

Some insights that can be derived from the dashboard include:

Distribution of employee salaries across departments

Performance comparison between employees

Workforce composition by gender and department

Identification of patterns that may influence HR strategies
