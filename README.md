Project Overview

This project focuses on analyzing HR data to uncover insights related to employee performance, salary distribution, and workforce demographics. The analysis was conducted in Microsoft Excel, emphasizing interactive dashboards and visualizations to support HR decision-making.

The goal is to demonstrate how Excel can be used as a powerful tool for data analysis, reporting, and insight generation in HR analytics.

Dataset Details

Number of records: 8,950 employees

Columns included:
Employee_ID, First Name, Last Name, Gender, State, City, Education Level, Birthdate, Hiredate, Termdate, Department, Job Title, Salary, Performance Rating

Data source: Kaggle

Data preparation steps:

Removed inconsistencies and duplicate records

Standardized data types and formats

Handled missing values

Structured data for pivot table analysis

This dataset allows for comprehensive analysis of HR trends, workforce composition, and performance evaluation.
